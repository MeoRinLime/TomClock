### 圆角按钮

```qml
QPushButton{
background-color: rgb(225, 225, 225);
border:2px groove gray;
border-radius:10px;
padding:2px 4px;
border-style: outset;
}

QPushButton:hover{
background-color:rgb(229, 241, 251); 
color: black;
}

QPushButton:pressed{
background-color:rgb(204, 228, 247);
border-style: inset;
}

```
